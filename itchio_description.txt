Pixel Clash.io

Pixel Clash.io est un jeu multijoueur rapide en arène (vue du dessus). Mangez des points pour grandir, tirez pour éliminer les autres, ramassez des power-ups et grimpez au classement.

Controls:
- ZQSD / W A S D : movement
- Mouse : aim
- Click : shoot

Monétisation: publicités intégrées (placeholders dans code). Remplace par AdInPlay/GameMonetize or CrazyGames.

Déploiement: client -> GitHub Pages / Netlify. Server (WebSocket) -> Render / Railway / Glitch.

Enjoy & share!
